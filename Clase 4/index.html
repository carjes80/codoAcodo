Formularios

Aqui empieza el formulario, con la etiqueta "form" y principalmente el atributo "action" se usa para definir la función del formulario al final, investigar

label es la etiqueta que se va a mostrar en el formulario, tiene el atributo "for" se usa para indicar el identificador del campo, estará asociado con el input en este caso es un texto o string

input Crea el campo para rellenar la consulta, tiene los parámetros "type" para el tipo de entrada, "id" para identificar el campo y linkear a css y "name" es cómo se asociará con la Base de Datos. Tambien hay "size", "maxlength", "required", "value" y otros más. Revisar la info del curso.

fieldset se usa para crear un cuadro y todo lo que se escriba aparecerá adentro. Tiene una etiqueta "legend" que se usa para poner el "titulo" del contenedor

select se usa para crear una lista desplegable, con la etiqueta "Option" te permite crear las listas

radio buttons Se usa para tener varios checkbox relacionados y que solo se pueda seleccionar 1. Se define de igual manera que los Checkbox
