Clase 1

Conceptos básicos de HTML

Ejemplo de cómo crear el código en HTML
<!DOCTYPE html>
<html>
<head>
<title>Título de la página</title>
</head>
<body>
<h1>Título principal</h1>
<p>Parrafo</p>
</body>
</html>

Se usó el editor (IDE) Visual Studio Code que facilita la codificación por los diferentes atajos que tiene, Recordar que la primera clase fue introductoria, se mostró conceptos básicos de HTML, hicimos el primer sitio que sorprendentemente era un "Hola mundo" y un segundo con la descripción de las etiquetas "head", "body", "h1", "h2" y "p" y otras más. Además de etiquetas, tambien encontramos atributos

Otra cosa importante que vimos es que las etiquetas tienen el formato de apertura con simbolos "<texto>" y el cierre con los simbolos "<texto/>"
Por ejemplo:

  •   <h1>Título principal </ h1>
  •   <h2>Título secundario </ h2>
  •   <p>Párrafo </ p>
      • ...
  • Existe excepciones, p.ej. la etiqueta de salto de línea no está codificada como un par de etiquetas de apertura y cierre. Es un elemento independiente o vacío, y está codificado como   <br>

Los comentarios en HTML dentro del editor se definen de la siguiente manera:

    <!-- todo lo que esté acá es un, puede estar en una linea o varias lineas-->

Clase 2

Continuación HTML (listas y enlaces)

Ejemplo de listas
<ul>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ul>
<ol>
<li>Coffee</li>
<li>Tea</li>
<li>Milk</li>
</ol>
<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>

En la clase se vieron los tipos de listas, como lo son:

  • Listas Desordenadas (ul)
  • Listas Ordenadas (ol)
  • Listas Dependientes (ol+ul)
  • Listas de Definición (dl)

Siempre que se declara el tipo, se deben "listar" los item con la etiqueta <li>item x</li> como se ve en el cuadro anterior. Para las listas de definición, se usa <dt> y <dd>

Enlaces de referencia para las Listas

  1. Manual web
  2. w3schools
    • Unordered Lists
    • Ordered Lists
    • Other Lists:
      • Description Lists
Ejemplos de Enlaces <a href="url (página web, imagen, archivo, lugar de la página)">Texto que va a aparecer en el enlace</a>

Enlaces en la misma pagina:

<a name="identificador de la etiqueta">Texto al que se va a apuntar</a>

...

<a href="identificador de la etiqueta al que se quiere ir" >Texto que haga referencia al enlace</a>

Existen varios tipos de enlaces:

    La etiqueta <a> tiene atributos importantes ref= donde va el url y otro es el target= que por lo general será "-blank" para navegar al destino desde una pestaña nueva.

  • Enlaces absolutos
    • Son aquellos que tienen la dirección completa de la url, incluyendo desde el https://...
  • Enlaces relativos
    • Son las que enlazan a las paginas webs internas. Si tuviesemos carpetas diferentes, los metemos ahí. Los articulos por lo general se meten en carpetas
  • Enlaces locales
  • Enlaces a correo
  • Enlaces a archivos
  • Enlaces con imágenes

Los ejemplos de los enlaces se pueden ver en la web de la pag 2. Lo importante es recordar que se pueden hacer referencias dentro de la misma pagina, primero definiendo
<a name="Nombre de la etiqueta para la referencia">"todo lo que se escriba acá será el lugar al que se va a apuntar"</a>
Luego se puede hacer el llamado con la ref
<a href="Nombre de la etiqueta para la referencia">"Texto que se va a mostrar en la pantalla con el enlace"</a>

En esta página estoy usando estos enlaces para mostrar referencias a páginas externas, ubicaciones en la página y otros

Clase 3

Multimedia y Tablas

Multimedia

Ejemplo de img e iframe

<img src ="Aqui va la URL, dirección de carpeta y archivo, etc..." height="numero en pixeles" alt="Este texto alternativo (tambien para invidentes)" >

<iframe width="560" height="315" src= "url del video" >Video de youtube</iframe>

<link rel="icon" type=" image/vnd.microsoft.icon" href= "/Resumen/img/favicon-16x16.png"

Muchos de los enlaces embebidos para videos, mapas, audio, apps y otras, salen directamente al compartir desde la página, sin embargo cualquier página o video puede ser compartido con el uso de la etiqueta iframe. Como se muestra en el cuadro anterior.

Hay algunas páginas que no se muestran, por limitaciones u otras razones que aun no las termino de ver todas.

Siempre recomiendan crear carpetas para guardar imágenes y recursos que va a presentarse en nuestra página web

Se puede anidar a los enlaces una imagen

El favicon (icono que aparece en la pestaña de la página) es un archivo .ico, el código es el que se ve en la tabla de ejemplo/p>

Tablas

Ejemplo de tablas 
        <table>
                <tr>
                    <th>Company
                    <th>Contact
                    <th>Country
                </tr>
                <tr>
                    <td>Alfreds Futterkiste
                    <td>Maria Anders
                    <td>Germany
                </tr>
                <tr>
                    <td>Centro comercial Moctezuma
                    <td>Francisco Chang
                    <td>Mexico
                </tr>
        </table>

...de acuerdo al código anterior, tendremos:

Company Contact Country
Alfreds Futterkiste Maria Anders Germany
Centro comercial Moctezuma Francisco Chang Mexico

Las tablas se definen por columnas e items, es una forma antigua de representar las cosas, actualmente se deben usar de otra manera, sin embargo lo que vimos en clase considera lo siguiente:

  • Las Etiquetas importantes son <table> para definir la tabla como principal, tiene atributos que pueden ayudar a visualizar de mejor manera la información como por ejemplo "border". Tambien tiene etiquetas <tr> (table row), <th> (table header) y <td> (table no se que con d pero es celda) Para mas info ver w3schools
  • Las tablas estructuran en la medida que se van creando columnas y celdas
  • Por ello existen atributos como "rowspan" y "colspan" que se usan para combinar las filas y las celdas, como se muestra en el ejercicio hecho en clase y que se muestra abajo
